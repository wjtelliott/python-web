# Python web server

## Contains

- Python REST directories
- Static served style.css
- Template rendering
- New fact form and redirects

| Route | Page |
| ----- | ---- |
| `/` | Welcome page |
| `/pets` | Pet index page |
| `/pets/:id` | Pet details page |
| `/facts/new` | Fact submit form |
| `/facts` | Fact index page |
| `*` | 404 Page |
